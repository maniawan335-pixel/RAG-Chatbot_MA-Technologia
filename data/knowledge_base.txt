PORTFOLIO PROJECTS

Project 1 — SkinIntel PetCare AI
An AI powered pet healthcare platform. Features include CNN based skin disease detection from uploaded images, online vet consultation via video call, medical history management, location based clinic finder, and secure payment integration. Built with React, Django, PostgreSQL, and Keras.

Project 2 — Imagify
An AI powered SaaS image generator. Users type a prompt and the system generates stunning images instantly using OpenAI API. Features include credit based usage system, Stripe payment integration, and user authentication. Built with React, Next.js, Node.js, and MongoDB.

Project 3 — RAG Chatbot
A multi turn AI chatbot with document understanding built using LangChain, Gemini API, and vector embeddings. Can answer questions based on uploaded documents and remembers conversation history.
